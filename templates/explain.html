Q & A

Q: Why I need to use GitHub account to login, why not just use my GitHub user name to match?

A: GitHub api has rate limit, so we need your GitHub api token to request your repos data.

Q: Why Jobder ask for read/write permission to my GitHub repos instead of read only?

A: Unfortunately, GitHub doesn't porvide public repos read only permission on their api right now(Surprise? Me too). You can also find the discuss here. We will never write any data on your GitHub repos.

Q: Why You built Jobder

A: When I contributed to django, I meet great people like Tim Graham and Simon Charette, they are both excellent engineers and always nice to the newbie. I'm sure they are great employees that every company searching for as well as everyone will love to work with them. Products just time to time, long live open source.
